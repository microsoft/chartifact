## Presets
Use presets to provide predefined state configurations.


```yaml presets
- name: Small & Red
  state:
    size: 2
    color: red
- name: Large & Green
  state:
    size: 10
    color: green
- name: Medium & Gray
  state:
    size: 7
    color: gray
```


```yaml slider
variableId: size
value: 5
label: 'Size:'
min: 1
max: 15
step: 1
```


```yaml dropdown
variableId: color
value: blue
label: 'Color:'
options:
  - red
  - green
  - blue
  - gray
```
