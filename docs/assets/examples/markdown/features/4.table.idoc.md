## Table
Use tables for displaying and interacting with tabular data.
### Basic Table


```json tabulator
{
  "dataSourceName": "sampleData"
}
```


### Selectable Table


```json tabulator
{
  "dataSourceName": "sampleData",
  "tabulatorOptions": {
    "autoColumns": true,
    "layout": "fitColumns",
    "maxHeight": "200px",
    "selectableRows": true,
    "rowHeader": {
      "formatter": "rowSelection",
      "titleFormatter": "rowSelection",
      "headerSort": false,
      "headerHozAlign": "center",
      "hozAlign": "center",
      "width": 40
    }
  }
}
```


### Table with Download Buttons


```json tabulator
{
  "dataSourceName": "sampleData",
  "tabulatorOptions": {
    "autoColumns": true,
    "layout": "fitColumns",
    "maxHeight": "200px"
  },
  "enableDownload": true
}
```
