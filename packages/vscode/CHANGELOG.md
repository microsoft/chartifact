# Change Log

## [1.0.2]

- Fixed command activation for copied idoc files with parenthesis numbering. Commands now support these file naming formats for both .md and .json files:
  - `file.idoc.md`
  - `file.idoc copy.md`
  - `file.idoc copy 1.md`
  - `file.idoc (1).md`
